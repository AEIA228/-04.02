# 3. Тест-кейсы

## Позитивные тест-кейсы

### TC01: Создание нового проекта
**Описание:** Создание нового проекта Tauri с базовой конфигурацией
**Шаги:**
1. Установить Tauri CLI
2. Выполнить команду `npm create tauri-app`
3. Следовать инструкциям установщика
4. Запустить проект командой `npm run tauri dev`
**Ожидаемый результат:** Создан новый проект с базовой структурой и запущено окно приложения

### TC02: Сборка приложения
**Описание:** Сборка приложения для текущей платформы
**Шаги:**
1. Открыть терминал в директории проекта
2. Выполнить `npm run tauri build`
3. Дождаться завершения сборки
**Ожидаемый результат:** Создан исполняемый файл в директории `src-tauri/target/release`

### TC03: Работа с окнами
**Описание:** Создание и управление множественными окнами
**Шаги:**
1. Создать новое окно через API
2. Изменить размер окна
3. Добавить кастомный заголовок
**Ожидаемый результат:** Окно создано и настроено согласно параметрам

### TC04: Системные уведомления
**Описание:** Отправка системного уведомления
**Шаги:**
1. Импортировать модуль уведомлений
2. Создать новое уведомление
3. Отправить уведомление
**Ожидаемый результат:** Показано системное уведомление

### TC05: Работа с файлами
**Описание:** Чтение и запись файлов
**Шаги:**
1. Создать текстовый файл
2. Записать данные в файл
3. Прочитать данные из файла
**Ожидаемый результат:** Данные успешно записаны и прочитаны
